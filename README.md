Money managging app which read pdfs and csv files to show visual reports to the user.

features:
1. The user is able to visualise their financial pattern over their separate accounts or as a whole, and ad tag/categories to transactions. 
2. The user's password is hashed and stored and the files are encrypted to ensure security
3. The user can delete files by clicking, by doing so, they can delete all the corresponding transactions
4. The user is able to upload their banks statement from major uk banks (its guaranteed to convert santander, monzo, lloyds, hsbc, and should theoretically work for other banks with minor tweaks) as pdf and csv.
5. The user is able to download their unified bank statement as csv, and also graphs and charts and png.
6. The user is able to delete the whole account.
